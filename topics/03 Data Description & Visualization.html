COMP20008 - Data Description & Visualization: Exam Notes

Describing Data Succinctly

Data can be described both mathematically and visually to provide a quick overview:

  • Mathematical Summary: Use statistical conventions to summarize data effectively.
  • Visual Summary: Visual illustrations (charts, graphs) provide a high-level overview and are easy to interpret compared to raw data tables.

Simple Descriptive Statistics

Some common methods used to describe data include:

  • Count (Frequency): Number of occurrences for each category in a dataset.
  • Min, Max, Range: Minimum, maximum values, and the range of data.
  • Quartiles and Percentiles: Dividing data into parts to understand its spread.
  • Mean, Median, Mode: Measures of central tendency to summarize data.
  • Variance and Standard Deviation: Metrics to understand the variability or spread in the dataset.

Data Visualization - Part I

Data visualization helps communicate insights effectively:

  • Purpose: Provides a quick snapshot of high-level information and allows easier interpretation of complex datasets.
  • Tools: Visualization tools such as bar charts, histograms, line graphs, and scatter plots help to represent data insights visually.

Importance of Visualization

Visualizations are crucial for the following reasons:

  • Quick Understanding: Enables viewers to grasp patterns, trends, and outliers quickly.
  • Communicating Results: Useful for sharing results with stakeholders in an easy-to-understand format.
  • Exploration: Facilitates data exploration, aiding analysts in spotting relationships and insights that may not be obvious from raw data alone.
