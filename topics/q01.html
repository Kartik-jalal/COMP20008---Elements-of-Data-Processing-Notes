COMP20008 - Regression and Classification: Exam Practice

Question 1

Which of the following is the best example of a regression problem?

  • Predicting how much a customer will spend at an online retailer in the next year based on their past purchase history.
  • Predicting which sports team will win an upcoming match.
  • Predicting whether a customer will purchase an item at an online retailer based on their browsing history.
  • Predicting the likelihood that an Instagram user will like a given photo.

Correct Answer: Predicting how much a customer will spend at an online retailer in the next year based on their past purchase history.

Explanation: This is a regression problem because it involves predicting a continuous numerical value (i.e., the amount of money). Regression problems focus on predicting quantities, whereas classification problems involve predicting categories or labels.

Practice Question 1

Which of the following is an example of a classification problem?

  • Predicting the price of a stock at the end of the day.
  • Predicting whether an email is spam or not spam based on its content.
  • Predicting the temperature in a city tomorrow.
  • Predicting the height of a tree given its age.

Correct Answer: Predicting whether an email is spam or not spam based on its content.

Explanation: This is a classification problem because it involves predicting a label (i.e., "spam" or "not spam"). Classification problems focus on assigning inputs to one of several predefined categories.

Practice Question 2

Which of the following tasks would be best suited for regression analysis?

  • Determining the probability that a user will click on an advertisement.
  • Predicting the number of visitors to a website next month based on historical data.
  • Classifying an image as either a cat or a dog.
  • Predicting whether a loan applicant will default on their loan.

Correct Answer: Predicting the number of visitors to a website next month based on historical data.

Explanation: This is a regression problem because the output is a continuous numerical value (i.e., the number of visitors). Regression is used to predict quantities, while classification is used for categorical outcomes.

Practice Question 3

Which of the following statements is true about regression problems?

  • They always involve predicting a discrete class label.
  • They are used to predict the relationship between variables by fitting a model.
  • They involve predicting probabilities of categorical outcomes.
  • They are used to group similar items together.

Correct Answer: They are used to predict the relationship between variables by fitting a model.

Explanation: Regression problems aim to predict numerical values by fitting a model to the data. This model can be used to understand and quantify the relationship between variables, making regression ideal for predicting quantities.

Summary

Understanding the difference between regression and classification problems is crucial in machine learning. Regression deals with predicting continuous quantities, while classification assigns labels to data points. Make sure to identify the nature of the problem before choosing the appropriate algorithm.
