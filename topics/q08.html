COMP20008 - Pearson Correlation and Data Analysis: Exam Practice

Question 8

Assume we want to understand why some apartments in Melbourne rent for more money than others. You have collected data for 200 apartments and their weekly rent. For each apartment, you have also collected 3 attributes, namely, the size of the apartment in square meters, the number of bedrooms in the apartment, and the house number of the street the apartment is in. We now want to use Pearson correlation coefficient between the rent and the 3 other attributes.

Part 1: Briefly explain what the Pearson correlation coefficient is. [1 mark]

Answer: The Pearson correlation coefficient, also referred to as Pearson's r, is a statistic that measures linear correlation between two variables X and Y. It ranges from -1 to 1, where 1 indicates a perfect positive correlation, -1 indicates a perfect negative correlation, and 0 indicates no correlation at all.

Part 2: Considering the 3 attributes (square meters, number of bedrooms, house number), please explain how reasonable it is to calculate the Pearson correlation coefficient for each one to understand the relationship with the weekly rent. [2 marks]

Answer: Square meters is a ratio feature, the number of bedrooms is an interval feature (with ordinal characteristics), and the house number is a nominal feature (with minor ordinal characteristics). Therefore, Pearson's r will be informative for square meters, as it is a continuous variable. It will also be reasonable for the number of bedrooms, though there may be ties, which Pearson correlation does not handle well. However, calculating Pearson's r for house numbers does not make sense, as house numbers are not a numerical feature with any meaningful linear relationship to rent.

Part 3: Using KNN Clustering - Is it better or worse if additional attributes are highly correlated with the existing 3 attributes? [2 marks]

Answer: It is generally better to choose features that are not highly correlated with each other. Including attributes that are highly correlated with the existing 3 attributes could lead to overfitting, reducing the robustness of the model and making it less generalizable to new data. This is because redundant information can compound errors and lead to an overemphasis on certain relationships. Using non-correlated features should make the model more robust and reduce potential errors.

Practice Question 1

Which of the following values of Pearson's correlation coefficient indicates the strongest relationship between two variables?

  • 0.85
  • -0.95
  • 0.5
  • -0.3

Correct Answer: -0.95

Explanation: A correlation coefficient of -0.95 indicates a very strong negative linear relationship, meaning as one variable increases, the other tends to decrease in a strong linear manner.

Practice Question 2

If Pearson's correlation coefficient between two variables is 0, what does that imply?

  • There is a perfect positive linear relationship between the variables.
  • There is no linear relationship between the variables.
  • The variables are independent of each other.
  • The variables have a perfect negative linear relationship.

Correct Answer: There is no linear relationship between the variables.

Explanation: A correlation coefficient of 0 indicates no linear relationship between the variables. However, they may still have a non-linear relationship that Pearson's r cannot capture.

Practice Question 3

Why is it not appropriate to use Pearson's correlation coefficient for categorical variables?

  • Because Pearson's r only measures linear relationships.
  • Because categorical variables cannot be ordered or have meaningful distances.
  • Because it can only be used for positive relationships.
  • Because it requires at least three variables.

Correct Answer: Because categorical variables cannot be ordered or have meaningful distances.

Explanation: Pearson's r measures the strength and direction of a linear relationship between continuous variables. Categorical variables do not have a natural order or meaningful distances, which makes Pearson's correlation inappropriate for such data types.

Summary

The Pearson correlation coefficient is a measure of the linear relationship between two continuous variables, ranging from -1 to 1. It is suitable for interval and ratio variables but not for nominal or categorical data. When building models, it's important to avoid using highly correlated features to reduce overfitting and improve model robustness.
