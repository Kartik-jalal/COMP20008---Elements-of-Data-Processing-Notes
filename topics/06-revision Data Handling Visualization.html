COMP20008 - Data Handling & Visualization: Exam Notes

Data Wrangling

Data Wrangling: The process of transforming raw data into a more usable format for analysis, including cleaning and handling missing values.

  • Handling Missing Values:
    • Deletion: Remove rows or columns with missing values if they are not critical.
    • Imputation: Replace missing values with statistical measures such as mean, median, or mode.
    • Example: In a dataset with missing values for income, impute using the median to avoid skewing results.
  • Data Normalization: Scale data to a common range, often between 0 and 1, to improve model performance.

Data Applications

Data Applications: Utilizing data to extract insights and drive decisions in various domains.

  • Customer Insights: Businesses use customer data to identify buying patterns and improve marketing strategies.
  • Predictive Analysis: Using historical data to predict future events, e.g., predicting customer churn.
  • Example: Retailers analyze transaction data to offer personalized product recommendations to customers.

Data Description & Visualization

Descriptive Statistics: Summarize key characteristics of data, such as mean, median, mode, variance, and standard deviation.

  • Mean (μ): Average value of a dataset.
    μ = (∑xi) / n
  • Standard Deviation (σ): Measures the spread of data around the mean.
    σ = √[(∑(xi - μ)2) / n]

Visualization Techniques: Graphical methods to represent data, making patterns easier to recognize.

  • Bar Charts: Used to display categorical data as rectangular bars.
  • Line Charts: Show trends over time by connecting data points with lines.
  • Box Plots: Visualize the distribution of data through quartiles and identify outliers.

Outliers & Visualization Techniques

Outliers: Data points that differ significantly from other observations and may indicate variability or errors in data.

  • Detection Methods:
    • Z-Score: Measures how many standard deviations a value is from the mean. Values with |Z| > 3 are often considered outliers.
      Z = (x - μ) / σ
    • IQR Method: Outliers are those beyond the range of [Q1 - 1.5*IQR, Q3 + 1.5*IQR].
  • Visualization for Outliers: Use box plots or scatter plots to easily identify outliers in a dataset.

Iris Dataset & Histograms

Iris Dataset: A well-known dataset consisting of 150 records of iris flowers, categorized into three species.

  • Features: Sepal length, sepal width, petal length, petal width.
  • Histograms: Used to visualize the distribution of each feature.
    • Example: Plotting petal length reveals different distributions for each species, helping with classification.

Use Case: Histograms provide insights into feature importance and can help determine which features are useful for distinguishing between classes.

Data Visualization & Handling Techniques

Advanced Visualization Techniques:

  • Heatmaps: Visual representations of data where values are depicted by color. Useful for identifying correlations in a dataset.
  • Pair Plots: Scatter plots of each pair of features in a dataset to identify relationships.

Data Transformation Techniques: Methods used to modify data to fit better within a model.

  • Log Transformation: Useful when data has a long tail distribution. Helps stabilize variance.
  • Min-Max Scaling: Transform data to fit within a fixed range, typically [0, 1].
    Xscaled = (X - Xmin) / (Xmax - Xmin)
